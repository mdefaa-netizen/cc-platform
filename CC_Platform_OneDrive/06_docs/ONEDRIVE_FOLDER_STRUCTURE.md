A consistent, organized folder system for all personal and professional files.

## Naming Convention Rules

| Rule | Example |
|------|---------|
| Numbered prefix on main folders for sort order | `01_zerwa`, `02_phd` |
| All lowercase with underscores (snake_case) | `cover_letters`, not `Cover Letters` |
| Dates in filenames use `YYYY_MM_DD` format | `2026_04_03_meeting_notes.docx` |
| No spaces or special characters | `tax_return_2025.pdf`, not `Tax Return (2025).pdf` |
| Descriptive but concise names | `resume_data_analyst_v2.pdf` |
| Version numbers at end if needed | `dissertation_ch3_v2.docx` |

## How to Set This Up in OneDrive

### Step-by-step (no commands needed):

1. **Go to** [onedrive.live.com](https://onedrive.live.com) in your browser
2. **Create the 5 main folders** first:
   - Click **New > Folder** and name it `01_zerwa`
   - Repeat for `02_phd`, `03_personal`, `04_education`, `05_jobs`
3. **Open each main folder** and create the subfolders inside it
   - Example: Open `01_zerwa`, then create `01_business_admin`, `02_clients_projects`, etc.
4. **Keep going deeper** вЂ” create the innermost folders as needed
5. **Drag and drop** your existing files from `C:\Users\mohamed.defaa` into the correct folders

### Tips:
- You do NOT need to create every subfolder right away вЂ” start with the main ones and add subfolders as you need them
- When saving new files, use the naming convention: `lowercase_with_underscores.ext`
- For dated files, put the date first: `2026_04_03_meeting_notes.docx` (this keeps them sorted by date)

## Security Reminders

- **Never share** `03_personal/01_identity_docs/ssn/` publicly
- **Enable** OneDrive Personal Vault for sensitive docs (IDs, SSN, tax returns)
- **Back up** your `02_phd/01_dissertation/` folder regularly
- **Do not store** passwords or secrets as plain text files
